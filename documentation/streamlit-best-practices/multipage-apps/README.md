Streamlit provides multiple ways to define multipage apps. Understand the terminology and basic comparison between methods.](https://docs.streamlit.io/develop/concepts/multipage-apps/overview) [**Define multipage apps with `st.Page` and `st.navigation`** \\

Place additional Python files in a `pages/` directory alongside your entrypoint file and pages are automatically shown in a navigation widget inside your app's sidebar.](https://docs.streamlit.io/develop/concepts/multipage-apps/pages-directory) [**Working with widgets in multipage apps** \\
