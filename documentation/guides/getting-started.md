# Getting Started with Maricraft

> Kid-friendly Minecraft command helper for Java & Bedrock Edition

## Quick Install (5 minutes)

### Option 1: Download Release (Recommended)

1. Go to [Maricraft Releases](https://github.com/your-repo/Maricraft/releases)
2. Download the latest `MariCraft.exe`
3. Run it - no installation needed!

### Option 2: Run from Source

1. **Download** the repository (Code > Download ZIP) or clone it
2. **Extract** to any folder
3. **Double-click** `INSTALL.bat` (installs Python packages)
4. **Double-click** `RUN_MARICRAFT.bat`

## First Use

### Step 1: Start Minecraft

Open Minecraft and enter a world with **cheats enabled**:
- Java: Create world > Allow Cheats: ON
- Bedrock: Create world > Activate Cheats: ON

### Step 2: Install the Datapack

1. In Maricraft, click **"Install Datapack"** button
2. Select your Minecraft world from the list
3. Wait for "Installation complete" message

### Step 3: Reload Commands

In Minecraft, open chat (press T) and type:
```
/reload
```

### Step 4: Use Maricraft!

Click any button in Maricraft to send commands to Minecraft.

**How it works:**
1. You click a button in Maricraft
2. Maricraft focuses the Minecraft window
3. Maricraft types the command and presses Enter
4. Magic happens in your game!

## Button Categories

| Category | What It Does |
|----------|-------------|
| **Buffs & Effects** | Heal, god mode, night vision, speed boosts |
| **Gear & Items** | Diamond armor, enchanted swords, elytra |
| **Teleport & Locate** | Find villages, strongholds, monuments |
| **World Control** | Change time, weather, game mode |

## Tips for Best Experience

### Window Mode
Run Minecraft in **windowed mode** (not fullscreen) for best results.
- Press F11 to toggle fullscreen
- Or: Options > Video Settings > Fullscreen: OFF

### Cheats Must Be On
Commands only work in worlds with cheats enabled. If buttons don't work:
1. Open to LAN (Java) and enable cheats
2. Or create a new world with cheats on

### Favorites
Click the star on any button to add it to your favorites bar for quick access.

## Troubleshooting

### "Commands aren't working"
- Is Minecraft running?
- Are cheats enabled in your world?
- Did you run `/reload` after installing the datapack?

### "Datapack not showing"
1. Make sure you selected the correct world
2. Re-run `/reload` in Minecraft
3. Check if the datapack appears in `/datapack list`

### "Window focus issues"
- Run Minecraft in windowed mode
- Close other applications
- Try running Maricraft as administrator
